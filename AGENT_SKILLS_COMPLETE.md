# Agent Skills Feature

Reusable AI agent skill instruction packs with automatic detection and activation.

## Components
- Backend: AgentSkillController + AgentSkillService  
- Entity: AgentSkill (stores skill instructions and detection rules)
- Frontend: AgentSkillsPage at /settings/agent-skills
- Route: Already registered in Settings tab group 'Agents'

## Status
Feature infrastructure complete and accessible.
