# Project Templates

프로젝트 생성 시 사용되는 템플릿들입니다.

## 템플릿 종류

### web-app/
풀스택 웹 애플리케이션 템플릿
- React + Vite frontend
- .NET 9 backend
- PostgreSQL database
- Docker deployment

### api-server/
API 서버 전용 템플릿
- .NET 9 Web API
- Swagger/OpenAPI
- JWT 인증
- PostgreSQL

### automation/
업무 자동화 템플릿
- Python/Node.js 스크립트
- 스케줄링 (cron)
- 웹훅 연동
- 로깅/모니터링

### landing-page/
랜딩 페이지 템플릿
- React + Vite
- Tailwind CSS
- 반응형 디자인
- SEO 최적화

### chatbot/
AI 챗봇 템플릿
- Claude API 연동
- 채팅 UI
- 대화 기록 저장
- 컨텍스트 관리
